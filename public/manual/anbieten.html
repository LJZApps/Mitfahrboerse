Mitfahrgelegenheit anbieten

Auf dieser Seite erfahren Sie, wie Sie eine Mitfahrgelegenheit anbieten können. Das Anbieten einer Mitfahrgelegenheit ist einfach und kostenlos.

Sie benötigen folgende Informationen:

  • Ihre Kontaktdaten (Name, E-Mail, optional Telefon)
  • Ihre Adresse oder den Startpunkt der Fahrt
  • Den Zeitraum, in dem die Mitfahrgelegenheit gültig ist
  • Optional: Informationen zu Kosten und weitere Details
1

Formular aufrufen

Um eine Mitfahrgelegenheit anzubieten, klicken Sie auf der Startseite oder in der Navigation auf den Button "Anbieten".

2

Persönliche Daten eingeben

Geben Sie Ihre persönlichen Daten in die entsprechenden Felder ein:

  • Vorname: Ihr Vorname
  • Nachname: Ihr Nachname
  • E-Mail: Ihre E-Mail-Adresse für Kontaktanfragen
  • Telefon (optional): Ihre Telefonnummer für direkte Kontaktaufnahme
  • Klasse (optional): Falls Sie einer Schule oder Einrichtung angehören

Ihre E-Mail-Adresse wird für Interessenten sichtbar sein. Verwenden Sie eine E-Mail-Adresse, unter der Sie erreichbar sind.

3

Adressdaten eingeben

Geben Sie die Adressdaten für Ihre Mitfahrgelegenheit ein:

  • PLZ: Die Postleitzahl Ihres Startorts
  • Ort: Der Ort, von dem aus Sie starten
  • Straße (optional): Die Straße, von der aus Sie starten

Je genauer Ihre Adressangaben sind, desto besser können potenzielle Mitfahrer einschätzen, ob Ihre Mitfahrgelegenheit für sie passt.

4

Gültigkeitszeitraum festlegen

Legen Sie fest, in welchem Zeitraum Ihre Mitfahrgelegenheit gültig ist:

  • Gültig ab: Das Datum, ab dem Ihre Mitfahrgelegenheit verfügbar ist
  • Gültig bis: Das Datum, bis zu dem Ihre Mitfahrgelegenheit verfügbar ist

Wenn Sie keine Daten angeben, wird standardmäßig ein Zeitraum von heute bis in 6 Monaten festgelegt.

5

Zusätzliche Informationen angeben

Optional können Sie zusätzliche Informationen zu Ihrer Mitfahrgelegenheit angeben:

  • Kostenbeteiligung: Informationen zu den Kosten für Mitfahrer
  • Weitere Informationen: Zusätzliche Details wie Fahrzeiten, Treffpunkte, etc.

Je mehr Informationen Sie angeben, desto attraktiver wird Ihr Angebot für potenzielle Mitfahrer.

6

Angebot erstellen

Klicken Sie auf den Button "Angebot erstellen", um Ihre Mitfahrgelegenheit zu veröffentlichen.

Überprüfen Sie alle Angaben vor dem Absenden. Besonders wichtig sind korrekte Kontaktdaten und Adressangaben.

7

Bearbeitungscode notieren

Nach dem Erstellen erhalten Sie einen Bearbeitungscode. Mit diesem Code können Sie Ihr Angebot später bearbeiten oder löschen.

Wichtig: Notieren Sie sich diesen Code unbedingt! Ohne den Code können Sie Ihr Angebot nicht mehr bearbeiten oder löschen.
