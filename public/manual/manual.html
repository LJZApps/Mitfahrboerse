PDF-Generator für das Mitfahrbörse-Handbuch

Mit diesem Tool können Sie eine PDF-Version des Mitfahrbörse-Handbuchs erstellen. Folgen Sie den Anweisungen unten, um Ihre eigene PDF-Version zu generieren.

Voraussetzungen

Um das PDF zu generieren, benötigen Sie eines der folgenden Tools:

  • wkhtmltopdf - Ein Open-Source-Tool zur Konvertierung von HTML in PDF
  • Chrome oder Chromium - Mit der Druckfunktion des Browsers
  • Firefox - Mit der Druckfunktion des Browsers

Methode 1: Mit wkhtmltopdf (Kommandozeile)

Wenn Sie wkhtmltopdf installiert haben, können Sie den folgenden Befehl verwenden:

wkhtmltopdf --enable-local-file-access --page-size A4 --orientation Portrait --margin-top 20 --margin-right 20 --margin-bottom 20 --margin-left 20 index.html anbieten.html suchen.html verwalten.html support.html manual.pdf

Kopieren Sie diesen Befehl und führen Sie ihn in der Kommandozeile im Verzeichnis public/manual/ aus.

Methode 2: Mit dem Browser

Sie können auch Ihren Browser verwenden, um das Handbuch als PDF zu speichern:

  1. Öffnen Sie jede Seite des Handbuchs in Ihrem Browser:
    • Einleitung
    • Mitfahrgelegenheit anbieten
    • Mitfahrgelegenheit suchen
    • Angebote verwalten
    • Support
  2. Drücken Sie Strg+P (Windows/Linux) oder Cmd+P (Mac), um den Druckdialog zu öffnen
  3. Wählen Sie "Als PDF speichern" als Ziel
  4. Passen Sie die Einstellungen an (Papierformat, Ausrichtung, etc.)
  5. Klicken Sie auf "Speichern"

Um alle Seiten in einer PDF-Datei zu kombinieren, können Sie ein PDF-Tool wie Adobe Acrobat oder einen Online-PDF-Merger verwenden.
